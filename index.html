Find Cities By Consonants

Hatched from a radio quiz question: Can you think of a US state which has the same kind and number of consonants as one of its cities? For example, Georgia would need a city name containing exactly 2 g's and one r.

When you have your guess, open the console and type in findAMatch("stateName", stateName); in camelCase if the state name is more than one word long (lowercase if only one word). The first parameter is a string of the state name and the second represents the array of cities.

For example: findAMatch("newHampshire", newHampshire);

There are only 4 states that qualify. See if you can find all 4!
